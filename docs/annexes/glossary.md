# Glossary

## Accession identifier

The business identifier used to track the ordered imaging event operationally.

## DICOM

The imaging standard used for acquisition, storage, and image metadata exchange.

## DICOMweb

The web-based DICOM service family used for query, retrieval, and storage in RESTful patterns.

## FHIR

The healthcare interoperability standard used here for computable radiology orders, workflow state, study metadata, reports, and conformance artefacts.

## IHE Scheduled Workflow

The radiology workflow profile family that coordinates order placement, worklist, acquisition, and result progression.

## ImagingStudy

The FHIR resource used to expose imaging study metadata aligned to the DICOM study.

## MY Core

The Malaysian base implementation guide reused in this repository for generic resource definitions.

## PACS

Picture archiving and communication system.

## RIS

Radiology information system.

## StudyInstanceUID

The DICOM unique identifier for the imaging study.
