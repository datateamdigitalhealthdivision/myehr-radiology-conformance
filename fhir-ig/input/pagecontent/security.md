# Security

## Baseline

The FHIR layer assumes OAuth2 and SMART on FHIR with sandbox-first implementation.

## Provisional placeholders

- SMART metadata: `https://sandbox-auth.myehr.kkmhub.moh.gov.my/.well-known/smart-configuration`
- FHIR sandbox base: `https://sandbox.myehr.kkmhub.moh.gov.my/fhir/r4`

## Reminder

Imaging endpoint trust, archive access control, and audit expectations are part of the wider national conformance repository and not only this FHIR package.
