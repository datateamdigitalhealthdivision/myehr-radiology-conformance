# Terminology Bindings Mapping Note

## Imaging orderables

- Preferred current basis: RadLex
- Secondary mapping support: SNOMED CT

## Findings and measurements

- Findings: SNOMED CT preferred
- Quantitative measures: LOINC with UCUM units

## Diagnoses and indications

- SNOMED CT, ICD-10, or ICD-11 depending on the reporting and classification use case

## Imaging metadata

- DICOM code systems for modality and technical metadata

## Policy note

No local MOH code systems are defined in this draft. Implementers should therefore design for externally governed code systems and future local hosting rather than local bespoke codes.
