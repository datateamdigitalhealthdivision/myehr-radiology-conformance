# ADR-003: Identifier Placeholder Policy

## Status

Accepted

## Context

Several national identifier domains and operational identifier formats are not yet final, but the repository still needs enough structure for sandbox implementation.

## Decision

Use clearly labelled provisional naming systems, markdown guidance, and example-only URIs where necessary. Do not hard-code unfinished identifier policy as if it were final national direction.

## Consequences

- vendors can build configurable systems now
- the repository stays honest about what remains unsettled
- later finalisation should require configuration change rather than redesign
